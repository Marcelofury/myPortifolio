## Portfolio Website

This repository powers my personal portfolio website — a modern, responsive single-page application showcasing my journey in Software Engineering, Data Analytics, AI, and Space Technology.

### Built With
- **Framework**: [Next.js 15](https://nextjs.org/) (App Router) with TypeScript
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Contact Form**: [EmailJS](https://www.emailjs.com/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Analytics**: Google Analytics (gtag.js)
- **Deployment**: [Vercel](https://vercel.com/)

### Features
- Modern glass-morphism design with dark space theme
- Fully responsive (mobile, tablet, desktop)
- Smooth scroll animations with Framer Motion
- Working contact form via EmailJS
- SEO optimized with dynamic sitemap
- Google Analytics integration

### Sections
1. **Hero** — Introduction with animated profile image
2. **About** — Personal background and mission
3. **Education** — Academic achievements and coursework
4. **Experience** — Work and leadership experience
5. **Skills** — Technical skills with proficiency levels
6. **Projects** — 8+ featured projects with descriptions
7. **Achievements** — Awards, hackathons, and community involvement
8. **Contact** — Contact form and social links

### Getting Started
```bash
git clone https://github.com/Marcelofury/portfolio.git
cd portfolio
npm install
npm run dev
```
Navigate to [http://localhost:3000](http://localhost:3000)
