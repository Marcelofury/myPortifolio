# Resume File

This directory should contain your `resume.pdf` file.

The portfolio website links to `/resume.pdf` for the download resume functionality.

**To add your resume:**

1. Place your PDF resume file in this directory as `resume.pdf`
2. The file is already in `.gitignore` so it won't be committed to the repository
3. When deploying, make sure to include your resume.pdf file in the deployment

**Note:** The resume.txt file is just a placeholder for development purposes.
